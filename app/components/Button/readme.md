### 功能说明
模态框，组建内使用的模态弹框

### index.js
#### Button

参数 | 说明 | 类型 | 默认值 | 是否必填
---|---|---|---|---
href | 按钮跳转连接 | string | - | no
className | 按钮自定义 className | string | - | no
style | 按钮的自定义样式 | object | {} | no
onClick | 按钮点击的回调 | () => void | - | yes
showtype | 按钮显示的样式类型 | number | 4 | no
children | 按钮中内容 | any | - | yes
parentStyle | 按钮的外层的样式 | object | {} | no
